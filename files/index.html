Administrador de Historial de Becas para SNTSA

Este sitio permite administrar la base de datos de los trabajadores del SNTSA y sus becarios, manteniendo un historial de los mismos.

Funciones

Subir datos a la base de datos

En esta sección se sube un archivo de Excel con todos los datos de los trabajadores y sus becarios para hacer el registro en la base de datos.

Generación de hoja de datos

En esta sección se sube un archivo de Excel con nombres completos de los trabajadores. Se detectan usuarios no existentes o duplicados y se genera un archivo de Excel con los datos completos de los trabajadores junto a el becario seleccionado.

Buscar trabajador

En esta sección se puede hacer búsquedas de los datos de los trabajadores, becarios y su historial.

Listar datos de los trabajadores

En esta sección se puede listar todos los trabajadores y sus becarios registrados en la base de datos.
