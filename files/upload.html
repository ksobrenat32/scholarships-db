Administrador de Historial de Becas para SNTSA

Regresar

Este sitio permite subir un archivo de Excel con datos de los trabajadores del SNTSA y sus becarios, para almacenar en la base de datos.

Flujo de trabajo

Para subir los datos de los trabajadores y sus becarios, sigue los siguientes pasos:

  1. Descarga el archivo de Excel de ejemplo aquí.
  2. Llena el archivo de Excel con los datos de los trabajadores y sus becarios.
  3. Sube el archivo de Excel con los datos de los trabajadores y sus becarios.
  4. En caso de existir cambios, selecciona los datos a actualizar
